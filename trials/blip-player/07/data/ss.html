SegmentSender

Consumes instructions and sends them to SegmentReceiver.

Instructions are of the form
(instruction, (parameters-tuple)), ...

An example (in python) is as follows:

    [
    ("publseg",     # publish segment instruction
        ("A/v1",    # segment name
         30,        # duration
         [ ("x", 5), ("y", 20),("z", 5)  ], # asset list
         20)),      # start at

    ("waitfor", 
        (5,)),       # sleep duration

    ("publseg", 
        ("B/v1", 
         30, 
         [ ("x", 5), ("c", 15),("d", 10) ], 
         50))
    ]
